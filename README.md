# HOW-TO-INSTALL-WINDOW-11
Installing Windows 11 is straightforward if your computer meets the minimum system requirements (TPM 2.0, Secure Boot, UEFI, and a compatible 64-bit processor). Microsoft recommends backing up your important files before you begin.

Procedure for Installing Windows 11
Check system requirements
Ensure your PC supports Windows 11.
Back up important files to an external drive or cloud storage.
Create a bootable USB
Download the Windows 11 installation media from Microsoft's official site:
Windows 11 Installation Media
Use the Media Creation Tool to create a bootable USB drive (at least 8 GB).
Insert the USB drive
Plug the bootable USB into the computer where you want to install Windows 11.
Boot from the USB
Restart the computer.
Press the Boot Menu key (such as F12, F9, Esc, or F2, depending on your computer).
Select the USB drive as the boot device.
Start Windows Setup
Choose:
Language
Time and currency format
Keyboard layout
Click Next.
Click Install Now.
Enter the product key
Enter your Windows product key, or click I don't have a product key if you plan to activate Windows later.
Choose the Windows edition
Select the edition that matches your license (for example, Windows 11 Home or Pro).
Accept the license terms and click Next.
Choose the installation type
Upgrade: Keeps your files and applications.
Custom: Performs a clean installation (recommended for a fresh start).
Select the installation drive
Choose the partition where Windows will be installed.
For a clean installation, you may delete old partitions (this erases all data), then select the unallocated space and click Next. Windows will create the required partitions automatically.
Wait for installation
Windows will copy files, install features, and restart several times.
Complete the initial setup
Select your country and keyboard layout.
Connect to the internet if prompted.
Sign in with your Microsoft account or create a local account (depending on your installation options).
Configure privacy settings.
Finish setup
Once the desktop appears:
Install Windows Updates.
Install device drivers if needed.
Install your applications and restore your files.
